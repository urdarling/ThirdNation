html转义字符

| HTML 原始码 | 显示结果 | 描述                   |
| ----------- | -------- | ---------------------- |
| &lt;        | <        | 小於号或显示标记       |
| &gt;        | >        | 大於号或显示标记       |
| &amp;       | &        | 可用於显示其它特殊字符 |
| &quot;      | "        | 引号                   |
| &reg;       | ®        | 己注册                 |
| &copy;      | ©        | 版权                   |
| &trade;     | ™        | 商标                   |
| &ensp;      |          | 半方大的空白           |
| &emsp;      |          | 全方大的空白           |
| &nbsp;      |          | 不断行的空白           |
